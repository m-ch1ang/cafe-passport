# Café Passport

A React Native app for discovering and checking into specialty coffee shops. Built with Expo, Supabase, and OpenStreetMap.

## Stack

- React Native
- Expo
- TypeScript
- Supabase
- React Navigation
- OpenStreetMap / Overpass API

## Setup

```bash
git clone https://github.com/michaelchiang/cafe-passport.git
cd cafe-passport
npm install
cp .env.example .env   # add your Supabase credentials
npx expo start
```
